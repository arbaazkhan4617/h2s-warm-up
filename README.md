# AI Cooking Assistant

A simple AI-inspired Cooking Assistant built for the H2S PromptWars Warm-Up Challenge.

## Features

- Select Diet Preference
- Enter Budget
- Enter Number of People
- Generate Meal Plan
- Grocery List
- Ingredient Substitutions
- Budget Feasibility
- Cooking To-Do List

## Tech Stack

- HTML
- CSS
- JavaScript
- Static JSON

## Project Structure

```
index.html
style.css
script.js
meals.json
README.md
```

## How to Run

```bash
python3 -m http.server 8000
```

Open:

```
http://localhost:8000
```

## Assumptions

- Uses static JSON data.
- No backend or database.
- Built only for the warm-up challenge.
